# Mini ERP System

A full-stack Enterprise Resource Planning (ERP) web application built with React, Node.js, Express, and PostgreSQL.

## 🚀 Features
- **User Authentication:** Secure JWT-based Login & Registration.
- **Customer Directory:** Manage customer records.
- **Inventory & Stock Management:** Real-time stock tracking with low-stock alerts.
- **Sales Challans:** Multi-item delivery challan generation with automatic stock deduction.

## 🛠️ Tech Stack
- **Frontend:** React.js, Axios
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
